What is ..

MDN Web Docs (tambien conocido como MDN - Mozilla Developer Network) es un proyecto de documentacion de estándares web (CSS, HTML y JavaScript), las tecnologías de Mozilla y otros temas de desarrollo.
Donde todos tienen permiso de añadir y editar contenido. No necesitas ser un programador o un experto en tecnología

What can do?

  • Fixing content bugs
  • Update and existing article.
  • Write an article to help people learn about the web
  • convert code samples to be "live"
  • Write or update an API reference
  • Localize MDN content
  • Promote MDN
